**STEPS**
--------
**Step 1 — Install dependencies**

BASH: sudo apt update
sudo apt install openbox firefox python3 python3-gi gir1.2-gtk-3.0

**Step 2 - Clone Repo**

BASH: git clone https://github.com/foreignbandss/KioskDE
cd KioskDE

**Step 3 - Install**

BASH: chmod +x install.sh
./install.sh

**Step 4 - Reboot**

BASH: reboot
